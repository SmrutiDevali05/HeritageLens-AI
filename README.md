# HeritageLens AI

HeritageLens AI is an AI-powered cultural preservation platform designed to digitize, analyze, and preserve ancient manuscripts and historical archives.

This project was developed as part of a college hackathon to address the problem of inaccessible historical manuscripts across libraries, archives, and institutional repositories.

---

# Problem Statement

Many ancient manuscripts and historical records remain difficult to access because of:

- Lack of digitization
- Poor metadata
- Untranslated scripts
- Limited searchability
- Fragmented archive systems

HeritageLens AI aims to transform static historical records into intelligent, searchable digital knowledge resources.

---

# Features

- AI-powered manuscript upload interface
- Historical manuscript preview
- Semantic archive search UI
- OCR-inspired workflow simulation
- AI processing animation
- Responsive modern frontend
- Glassmorphism UI design
- Dark luxury theme
